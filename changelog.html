Version 1.0.0 August 22, 2025
Initial Release
  • Launched Atlas geography learning platform
  • Six different game modes: Population, Playground, Time, Hint, Football, and Duel

  • Responsive design for all devices
  • Basic analytics and performance tracking

  • Implemented global leaderboard with Supabase integration
  • Added player flag selection system
  • Created interactive world map with country image overlays

Note: This changelog is updated with each major release. For detailed technical changes, check our development repository.
